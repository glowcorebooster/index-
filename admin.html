📖 Setup Guide — Isang beses lang ito!
1
Pumunta sa Google Sheets → gumawa ng bagong spreadsheet → pangalanan itong GlowCore Products
2
I-copy ang Spreadsheet ID mula sa URL (yung mahabang text sa gitna ng URL)
3
Sa Google Sheets, click Extensions → Apps Script
4
I-paste ang Apps Script code mula sa SETUP.md file na kasama sa ZIP
5
Click Deploy → New Deployment → Web App → set "Who has access" to Anyone → Deploy
6
I-copy ang Web App URL → i-paste sa input field sa taas ng form
